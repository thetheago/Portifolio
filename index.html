Sobre

Olá, bem vindo ao meu currículo! 😀 Sou Thiago Henrique, tenho 19 anos e decidi fazer esse CV usando HTML / CSS / JS e PHP para mostrar meu conhecimento e habilidade no assunto! Separei os arquivos com o código escrito lá no meu github, é possível acessa-lo clickando no botão ao lado IR PARA GITHUB ou se quiser acessar meu Github completo é só clickar no "Acesse meu github" no topo da página.

Estou atualmente cursando o 3° semestre em Ciência da Computação na faculdade Fumec. Estágiando na empresa CDR Soluções Logística, onde minhas príncipais funções são voltadas para progamação em PHP e interação com banco de dados MYSQL, outras funções que exerço bastante na empresa são voltadas para HTML, CSS e JS, fico pesquisando diferentes formas de deixar o layout mais agradável da melhor maneira possível, acabei gostando bastante de CSS e Javascript pois antes de trabalhar com programação eu trabalhava freelance e por hobbie com edição audiovisual então ví semelhanças quando comecei a enteder CSS e as possíveis animações com JS. Então decidi fazer esse CV com essa animação bem legal no header usando o JS + JQuery para adicionar margin as 3 imagens conforme o scroll é rolado!
